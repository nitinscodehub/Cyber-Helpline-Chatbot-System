# 🚨 Cyber Helpline Chatbot System

An AI-powered chatbot application designed to assist victims of cybercrime by providing **instant guidance, reporting assistance, and awareness resources**.

This project was presented at **Smart Solutions Exhibition 2.0 – Ekatva 2026** at the **Central University of Jammu**.

# 📌 Project Overview

Cybercrime victims often panic and do not know the correct steps to take after an incident.

The **Cyber Helpline Chatbot System** helps users by:

* Identifying the type of cybercrime
* Providing immediate response guidance
* Suggesting reporting procedures
* Spreading cyber awareness

The system uses **Natural Language Processing (NLP)** to understand user queries and respond intelligently.

---

# ✨ Key Features

🕒 **24/7 Availability**
Users can access the chatbot anytime.

🔍 **Cyber Incident Classification**
Detects cases like:

* Phishing
* Online fraud
* Identity theft

⚡ **Immediate First-Aid Guidance**
Provides steps victims should take immediately.

🌐 **Multi-Language Support**
Supports multiple languages for accessibility.

📢 **Cybercrime Reporting Support**
Guides users on how to report incidents.
